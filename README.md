# Notes-App-ReactJS

A beautiful and responsive notes application built with React and Tailwind CSS. Create, view, and delete notes with an elegant sticky-note UI.

## Features

- ✨ Create notes with title and detailed content
- 📝 Clean and intuitive interface
- 🗑️ Delete notes with a single click
- 📱 Responsive design for all devices
- 🎨 Beautiful sticky-note themed UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript ES6+

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
